This test suite have been Built for FaMaPY, a tool develop in Python 3 to the automated analisys of features models. The intention of 
this suite was check the right funcionality of the operations implemented in Glucose3 PySat solver.

Before run the test suite, you need to have installed FaMaPy and his metamodels modules, and all the dependencies needed. 

For run the test you only need run TEST.py, this file will be responsible of call the test classes with all the test cases in there.
